    <!DOCTYPE html>

    <!doctype html>

  6. article 和 section 的区别

    article元素代表文档、页面或者应用程序中独立完整的的可以被外部引用的内容。因为article元素是一段独立的内容，所以article元素中，通常包含头部、底部。

    section元素用于对网站或者应用程序中页面上的内容进行分块，相邻的section元素的内容应当是相关的，而不是像article那样独立。一个section元素通常由内容以及标题组成。

第三部分

  1. 为什么不建议通过 HTML 设置显示状态？

    html   主要用于页面结构搭建，通常称为结构层
    css    设置样式，称为表示层
    js     业务处理，称为行为层

  2. 你的 HTML 代码风格是什么？为什么这样做？

    html 中css和js都采取外部引入，这样代码会保持清晰地结构。

    在js中需要操作dom的时候，都会对其定义一个以js-**为开头的类名，这样保证css中使用的class改变，但是不会对js业务产生影响。

  3. 如何确定浏览器对 HTML5 的支持？

    在html中写一些html5的标签去检测是否支持，使用js方法也可以检测。
